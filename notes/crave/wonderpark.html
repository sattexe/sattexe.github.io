Selfie World(TV)

  • TV (No signal)
  • Check the HDMI connection in the PC and also TVs
  • Check the INPUT SOURCE in the TV - make sure the HDMI no in Input Source matches cable connection behind the TV. For example, in TV, the HDMI cable is connected to the HDMI-3 port, means the INPUT SOURCE also must be HDMI-3
  • App not full screen
  • Close the app by pressing ALT + F4
  • Open the ResolutionFix.bat in the Desktop
  • Now you can open back the application / Just restart the PC
  • The character is not loaded
  • Restart the PC
  • Or open this link from browser : http://localhost/Clear.php
  • If it’s still the same, please contact Crave Tech Support
  • Application crash
  • Restart PC

  1. COLOR ALIVE

Color Alive World (Projectors (Wall))

  • Projector(No signal)
  • Check the HDMI connection in the PC and also Projectors
  • Check the INPUT SOURCE in the Projector make sure the HDMI no in Input Source matches cable connection on the projector. For example, in Projector, the HDMI cable is connected to the HDMI-1 port, means the INPUT SOURCE also must be HDMI-1 (Use Panasonic & EPSON black remote)
  • App not full screen
  • Close the app by pressing ALT + F4
  • Open the ResolutionFix.bat in the Desktop
  • Now you can open back the application / Just restart the PC
  • App loads into Single screen / Surround is not enabled
  • Right click on Desktop, and open NVIDIA Control Panel
  • Click on Configure Surround, PhysX, check the Span displays with Surround and then click Configure
  • Sort the displays according to the numbers shown. And then click Enable Surround.
  • Make sure all three projectors are selected
  • Wau not loaded in the World
  • Close the app by pressing ALT + F4
  • Find the ColorAlive app shortcut in the Desktop, right click and choose Open File Location.
  • Go into ColorAliveWorld_Data/StreamingAssets
  • Check the FilePath > C:\xampp\htdocs\Images\ColorAlive\
  • Check the ServerAddress > 192.168.0.111
  • Application crash
  • Restart PC
